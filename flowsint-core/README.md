# flowsint-core
The repository containing the core components for flowsint.

It contains:
- Celery tasks
- Authentication
- NEO4J database connectors
- POSTGRESQL database connectors
- Enricher base class
- Enricher orchestrator
- Enricher registry
- Encrypted vault

## Testing

```bash
poetry run pytest
```
